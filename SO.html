Windows

É o sistema operacional para computadores mais utilizado no mundo, as versões mais utilizadas são windows 7 e windows 10. A razão por ser mais utilizados é pelo fato de ser bem completo e ter acesso a tudo. Um lado negativo que atualmente passamos e não é nada animador é que a pirataria dos windows está em uma escala assustadora (não encontrei nenhum lugar que mostre o quanto de pessoas não ativaram o windows), só não está pior a situação porque notebooks que ja vem de fabrica com o windows já vem com uma chave ativa gravada na propria placa mãe, ou seja, você poderá reinstalar o windows quantas vezes você quiser, sempre irá funcionar a chave de ativação e o melhor que é de graça e é legal.

Podemos destacar tambem, existem dois tipos de windows, 32 bits e 64 bits, se você tem uma cpu 32 bits coloque um windows 32 bits e se por acaso você tiver uma cpu de 64 bits você pode optar por 32 bits ou 64 bits já que não tem problema por que a cpu tem compatibilidade com os 2, mas mesmo assim coloque um de 64 bits é melhor

O Windows original na loja da proprietaria Microsoft está custando R$1.099,00 (versão Windows 11). Como quase nenhum brasileiro tem condições de comprar o software original as pessoas pirateiam colocando "crack" para gerir uma chave aleatória.

Linux

É um sistema operacional open source, ou seja, codigo aberto onde qualquer pessoa pode alterar e consequentemente não é preciso de licença para poder usar (gratuito), por ser um sistema operacional simples ele acaba sendo uma melhor maneira de ressuscitar um notebook ou um computador antigo que não está lidando com tarefas simples do dia a dia, consome pouquissimo processamento e pouquissima memória RAM

Mac OS

O Mac provavelmente é um sistema operacional que bate de frente com o Linux por ser proprietaria da Apple fica dificil colocar em outros computadores, quando algum computador que não seja da apple tem o sistema operacional Mac OS nós falamos que a pessoa fez um "Hackintosh", pelo simples fato da empresa só aceitar o sistema operacional em computadores autorizados (que seja da marca Apple basicamente). As versões mais conhecidas são Catalina, Big Sur e Monterey e tambem as mais estaveis.

API's

Bem, ja que falamos de Sistemas Operacionais (Software) e de Hardware vamos nos aprofundar um pouco em API's que são nada mais nada menos do que uma interface que liga o Hardware com o Software (Application Programming Interface). Com elas podemos citar vários exemplos de API como:

DirectX: API gráfica que gerencia uso de processador, memória RAM e Placa de Vídeo (as mais utilizadas são as versões DX11 e DX12).

OpenGL: API gráfica que basicamente faz a mesma coisa que foi citado acima mas... funciona quase que com exclusividade com jogos com interfaces gráficas leves visto que tambem gerencia uma parte das texturas fazendo a manipulação das libs e frameworks de uma maneira otimizada.

Vulkan: API gráfica da mesma empresa da OpenGL mas feita para substituir a tecnologia passada. O foco dela é extrair o melhor dos núcleos e das instruções do processador. Não é a toa que vimos muitos jogos como Rainbow Six Siege, Dota 2, Fortnite, Red Dead Redemption, Rise of the Tomb Raider entre outros. É uma lista muito grande e todos eles buscam apenas uma coisa, o maximo de desempenho. Apesar de ser eficiente na maioria dos casos não são todos os games que sai bem quando falamos de desempenho, muitas vezes é por que o jogo demanda muito processamento grafico e não seja uma boa pedida pôr vulkan, talvez seja melhor o DirectX 12.
