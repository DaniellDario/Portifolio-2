VRAM

Funciona de forma parecida com a memoria ram, mas, é especifica para memoria de vídeo. Toda informação pertencente a vídeo, nela podemos encontrar textura, informações de driver, informações do monitor e etc.
Elas são compostas por varios "chips" de memoria se podemos assim chama-lo, são essas memorias que vão dizer a velocidade de banda da memoria, ou seja, para não ficar muito complexo para entender;

Imagem: Placa de Vídeo aberta

Essas memórias cada uma tem uma interface de comunicação de 32 bits, atualmente placas de video atuais como 3060ti da Nvidia tem 8gb de memoria (256 bits) ou seja 32x8 (8 chips de memória de 1gb cada), quanto mais bits melhor para largura de banda.

Tipos de Memória

A memória é a GDDR basicamente graphics double data rate synchronous dynamic random-access memory, ufa!, o nome é grande mas basicamente serve como uma memoria ram de acesso mais rapido para a placa grafica e ponto. A tecnologia mais recente é a GDDR6X, para efeito de comparação das memorias te recomendo a assistir este video.

Video: Comparação de Memória GDDR6 vs GDDR6X

Clock

Basicamente a mesma coisa que o clock faz no processador, que é quantos ciclos faz por segundo, tanto para o processador gráfico quanto para a memória que está embarcada na placa de vídeo. O clock das placas de videos geralmente não tem tanta alteração se irmos comparar com ás de um processador, mas, se compararmos a memoria RAM com a memória de vídeo (VRAM) a diferença de clock é absurda é quase o dobro de velocidade (placa de video vencendo por velocidade).

Ray Tracing

Como vimos na primeira imagem o Ray tracing é uma tecnologia baseada em inteligência artificial, que busca entender como os traçados de raios de luz funcionam, somente placas de video com aceleradores de traçados de raios podem fazer isso, como é o caso das RTX da NVIDIA.
